# 场景编辑

打开 场景编辑 时界面默认打开的是一个新的工程可以直接在当前界面上进行工程的创建。

# 一、新建工程

通过 新建 按钮一个空白全新的工程覆盖原本存在显示界面上的工程（不进行保存操作）

# 二、打开工程

通过 打开 按钮可以选择该用户账号下所有已创建的工程文件进行打开操作(该列表中可对工程进行删除操作)，选中的工程文件显示在场景编辑界面上，可以对其进行编辑，新增，删除等操作

# 三、工程配置

  1. # 场景选择

    # 添加场景

    在工程下方进行 添加场景 选择 全景场景图/ 图片场景图/ 模型场景图进行添加。

    • 全景图场景：通过选择布局模板上传的布局模板，再选择布局模板中的全景图素材或者六面体素材（六面体素材会组合成一张全景图样式进行添加）进行添加操作

    • 图片场景图：通过选择 素材管理 中的图片素材进行添加操作

    • 模型场景图：通过选择 素材管理 中的模型素材进行添加操作

    ①当不进行选择场景类型时，点击 添加场景 按钮，默认为添加全景图场景素材

    ②一次性可以选择同类素材多张进行添加，素材排列的顺序和素材在布局模板/素材管理中的顺序有关与选择素材时的顺序无关

    # 场景顺序

    添加过后的场景可以编辑顺序，通过下方场景缩略图左上角三个点的编辑按钮可以选择四种顺序变化

    • 前移：该场景图移动到前一个场景图之前

    • 后移：该场景图移动到后一个场景图之后

    • 移到最前：该场景图移动到该工程所有场景图之前

    • 移到最后：该场景图移动到该工程所有场景图之后

    # 场景编辑

    • 场景删除：点击场景图缩略图左上角编辑按钮可以选择删除按键，界面出现确认删除弹窗选择确认后该场景图被删除（包括该场景图上的所有设置，热点等）
    • 场景命名：点击场景缩略图下名称的编辑按钮，可以对场景名称进行自定义命名（如果不对其进行命名，系统会按照场景图添加的顺序命名为 场景x ，x代表自然数）

    注意：进行场景命名时，蓝色选择框一定要出现在编辑命名的场景缩略图上，否则无效

    • 场景语音：点击任意一个场景缩略图后，场景图右侧的 基础设置 中的下方有 场景设置 区域，语音讲解 模块是一个文本框，填入该场景的语音解说词，当工程完成后点击进入该场景时会同步播放所填入的语音解说词。当内容为空时，系统默认该场景没有语音解说词不进行播报。
    • 场景更换：场景设置 中 场景模板 模块可以进行场景图更换，点击其中 选择模板 可以进行场景图的替换

    注意：场景的替换只能是同类型场景图的替换，图片场景图上选择模板只能选择素材库中图片素材，全景场景图和模型同理

  2. # 基础配置

    # 工程名称

    该功能模块用于编辑工程名称

    注意：当一个新的工程进行工程命名后进行保存操作，界面仍然会弹出工程命名的输入框

    # 加载页面:

    点击 加载页面 进入加载页面配置，该界面出现 加载图标 和 重置 两个功能按钮，点击前者可以从素材库图片类型中选择一张图片作为进入工程前的加载页面

    # 虚拟人物:

    当工程进行场景语音播报或者热点语音播报时，虚拟人物形象会随着语音播报出现同步出现，随着语音播报结束而消失

    点击 虚拟人物 进入虚拟人物页面配置，界面上将虚拟人物功能分为 人物设置 和与 语音设置

    • 人物：点击 选择人物 可以从素材库的虚拟人物中选择合适的虚拟人物形象

    • 位置：可以选择虚拟人物出现在工程界面上的位置，只能在 左下 和 右下 两个选项中进行选择

    • 大小：对虚拟人物图片的 宽度 和 长度 进行配置来调整虚拟人物的大小

    • 讲解声音：可以在下拉框中选择喜欢的语音讲解声音，该讲解声音暂不开放自定义添加

    • 讲解音量：通过拉动0-100范围的数值来配置讲解音量大小，默认值为50

    • 讲解语速：通过拉动0-100范围的数值来配置讲解语速快慢，默认值为50

    注意：虚拟人物的配置属于全局配置，是指该配置属于工程配置出现在所有的场景中，并不需要每个场景都进行配置

  3. # 视角配置

    视角设置能够限制场景视角范围，使场景进行观览时保持在你设置的视角范围内，最大可以实现360°全景观览

    该视角配置主要效用于全景图场景和模型场景，对图片场景影响不大

    # 恢复默认视角

    默认视角为场景图的初始视角，即图片的默认视角（添加场景图时，显示的即为初始视角）

    # FOV视角范围

    FOV视角是指你与场景图之间的焦距视角，即你可以看到的最近视角到最远视角范围，该视角分为 最近 ，初始 ，最远 三项。默认值分别为 2 ，45 ，120
    最近数值的极限值为1，最远数值的极限值为179

    • 最近：最近数值表示你与场景图最大焦距，数值越小焦距越大，表示视野越小，离场景图就越近

    • 初始：表示你进入该场景时的焦距，初始值应该在一个合适的数值上，保证场景信息的正常阅读

    • 最远：最远数值表示你与场景图最大焦距，数值越大焦距越小，表示视野越大，离场景图就越远

    注意：最近数值不能大于最远数值（系统已做了限制），初始值可以等于最近数值或者最远数值，当需要更改该项数值时，应当从最远数值，初始数值，最近数值按顺序进行配置

    # 垂直视角范围

    垂直视角指能够通过上下滑动对场景上方区域以及下方区域进行观览，该视角分为 最低 和 最高 ，默认值分别为 -90 ，90
    当最低数值为-90，最高数值为90时，已达到度数最大值。

    • 最低：指能够往初始视角下方看见的最大角度，这里的数值填写和度数相关，默认初始视角为水平线度数为0°

    • 最高：指能够往初始视角上方看见的最大角度，这里的数值填写和度数相关

    # 水平视角范围

    水平视角指能够通过左右滑动对场景对视角水平区域进行观览，该视角分为 最低 和 最高 ，默认值分别为 -180 ，180
    最低数值的极限值为-360，最高数值的极限值为360；当最高数值减去最低数值等于360时场景就会实现360°水平观览 *注意：当数值为360°时相当于0°；当最低数值为0，最高数值为360或者最低数值为-360，最高数值为360时会造成视角无法水平转动。

    • 最低：限制初始视角向右侧的最大水平角度，这里的数值填写和度数相关，默认初始视角为水平线度数为0°

    • 最高：限制初始视角向左侧的最大水平角度，这里的数值填写和度数相关

    # 设置当前为初始视角

    当功能栏在视角设置中时，界面上出现 设置当前为初始视角 的文字框，移动场景视角到任意位置，点击该文字框后，初始视角就会变成当前场景视角且FOV视角中初始值会变成当前视角的FOV数值

  4. # 热点配置

    所谓“热点”就是配置悬浮在场景图上方的功能按钮，用于展示场景内产品信息或者实现场景切换等功能，热点可以进行图标配置和内容配置

    # 添加热点

    点击 增加热点 按钮，热点图标会默认新建在当前场景图正中间，并且右侧功能区弹出热点配置界面

    • 选择图标：选择图标分为 预设图标 和 自定义图标 。

    ①预设图标可以在系统已存在的各种图标中选择其中一项，点击热点配置 确认 后生效

    ②自定义图标可以点击 选择图片 ，界面弹出素材库图片列表，选择其中一项后点击热点配置 确认 后生效

    • 标题名称：标题名称即热点名称，进行文本框内容填写后，点击热点配置 确认 后生效，当取消勾选 显示标题 栏时，场景图上该热点的标题名称会消失但是在热点列表中不会消失（当勾选 显示标题 后，使标题名称的文本框内容为空后，场景图上该热点图标上会显示一个小灰点）
    • 热点类型：热点类型分为十种，热点类型不同配置内容不同,如下所示

    ①场景切换：场景切换包括 选择目标场景 和 场景水平视角

    选择目标场景：可以通过下拉框选择当前工程所有场景图中任意一个场景图

    场景水平视角：该功能可以进行选择勾选使用，当未勾选时表示使用目标场景的初始视角作为进入该场景的第一视角；当勾选时可以在文本框中填写数值进行场景视角的配置，该数值表示与目标场景初始视角的度数差（以目标场景初始世家为0°），正数往目标初始视角右边移动，负数往目标初始视角左侧移动。

    ②文字热点：文字热点包括 文字内容 ，宽高单位 ，位置 ，语音讲解

    文字内容：可在文本框中填写入任意文本内容用于显示在场景图之上，字数暂无上限

    宽高单位：宽高单位分为两种方式，一种是 像素 ，一种是 百分比

    像素：通过调整 宽度 和 高度 的数值来放大或者缩小素材显示区域（当宽度为1600，高度为800时，显示页面填满整个场景图），默认值分别为 480 和 320
    注意：若是宽度和高度超过全屏数值会造成热点返回场景按钮超出屏幕之外无法点击

    百分比：通过调整 宽度 和 高度 的百分比数值来放大或者缩小素材显示区域（当宽度为100，高度为100时，显示页面填满整个场景图），默认值分别为 50 和 50
    注意：宽度和高度的最小百分比数值为1，最大百分比数值没有做出限制，但是即使百分比数值超过100，显示的最大数值仍然为100（例如宽高百分比数值为200但是只有100的显示效果）

    位置：位置表示素材显示区域相对于热点图标的相对位置，分为12个选项分别为 左上 ，中上 ，右上 ，中左 ，正中 ，中右 ，左下 ，中下 ，右下 ，左侧 ，右侧 ，全屏 ；可以根据你所创建的场景进行合适的配置

    语音讲解：语音讲解框中可以填写任意文字内容，用于与热点展示时同步展示语音，多用于产品介绍等多种场景

    ③图片热点：图片热点包括 图片内容 ，宽高单位 ，位置 ，语音讲解

    图片内容：点击 选择图片 可从素材库图片类型中选择一张或者多张图片素材（当选择多张图片素材时，系统默认图片循环轮播，时间间隔为5秒）

    宽高单位：宽高单位分为两种方式，一种是 像素 ，一种是 百分比

    像素：通过调整 宽度 和 高度 的数值来放大或者缩小素材显示区域（当宽度为1600，高度为800时，显示页面填满整个场景图），默认值分别为 480 和 320
    注意1：若是宽度和高度超过全屏数值会造成热点返回场景按钮超出屏幕之外无法点击
    注意2：当像素大小大于或者小于原本的素材时，保持素材正常高宽比的同时多余的像素面系统会自动使用黑色像素代替

    百分比：通过调整 宽度 和 高度 的百分比数值来放大或者缩小素材显示区域（当宽度为100，高度为100时，显示页面填满整个场景图），默认值分别为 50 和 50

    注意2：当百分比宽高比与原本的素材宽高比不同时，保持素材正常高宽比的同时多余的像素面系统会自动使用黑色像素代替

    位置：位置表示素材显示区域相对于热点图标的相对位置，分为12个选项分别为 左上 ，中上 ，右上 ，中左 ，正中 ，中右 ，左下 ，中下 ，右下 ，左侧 ，右侧 ，全屏 ；可以根据你所创建的场景进行合适的配置

    语音讲解：语音讲解框中可以填写任意文字内容，用于与热点展示时同步展示语音，多用于产品介绍等多种场景

    ④视频热点：视频热点包括 视频内容 ，宽高单位 ，位置 ，语音讲解

    视频内容：点击 选择视频 可从素材库视频类型中选择一个视频素材

    宽高单位：宽高单位分为两种方式，一种是 像素 ，一种是 百分比

    像素：通过调整 宽度 和 高度 的数值来放大或者缩小素材显示区域（当宽度为1600，高度为800时，显示页面填满整个场景图），默认值分别为 480 和 320
    注意1：若是宽度和高度超过全屏数值会造成热点返回场景按钮超出屏幕之外无法点击

    语音讲解：语音讲解框中可以填写任意文字内容，用于与热点展示时同步展示语音，多用于产品介绍等多种场景（如果该视频素材原本存在声音，该功能建议不使用使其文本框内容为空）

    ⑤文档热点：文档热点包括 文档内容 ，宽高单位 ，位置 ，语音讲解

    文档内容：点击 选择文档 可从素材库文档类型中选择一个文档素材

    H5内容：点击 选择H5 可从素材库H5类型中选择一个H5素材

    宽高单位：宽高单位分为两种方式，一种是 像素 ，一种是 百分比

    像素：通过调整 宽度 和 高度 的数值来放大或者缩小素材显示区域（当宽度为1600，高度为800时，显示页面填满整个场景图），默认值分别为 480 和 320
    注意1：若是宽度和高度超过全屏数值会造成热点返回场景按钮超出屏幕之外无法点击
    注意2：当像素大小大于或者小于原本的素材时，保持素材正常高宽比的同时多余的像素面系统会自动使用黑色像素代替

    百分比：通过调整 宽度 和 高度 的百分比数值来放大或者缩小素材显示区域（当宽度为100，高度为100时，显示页面填满整个场景图），默认值分别为 50 和 50
    注意1：宽度和高度的最小百分比数值为1，最大百分比数值没有做出限制，但是即使百分比数值超过100，显示的最大数值仍然为100（例如宽高百分比数值为200但是只有100的显示效果）
    注意2：当百分比宽高比与原本的素材宽高比不同时，保持素材正常高宽比的同时多余的像素面系统会自动使用黑色像素代替

    位置：位置表示素材显示区域相对于热点图标的相对位置，分为12个选项分别为 左上 ，中上 ，右上 ，中左 ，正中 ，中右 ，左下 ，中下 ，右下 ，左侧 ，右侧 ，全屏 ；可以根据你所创建的场景进行合适的配置

    语音讲解：语音讲解框中可以填写任意文字内容，用于与热点展示时同步展示语音，多用于产品介绍等多种场景

    • 确定/删除：点击热点配置中 确认 按钮，以上热点配置的内容才会生效，否则默认取消；点击热点配置中 删除 按钮会移除该热点。

    # 移动热点

    只有在热点配置栏目下才能够移动热点，在场景图上拖拽热点进行场景图上的位移，移动到指定的位置后松开鼠标左键即可生效。

    # 热点列表

    热点列表显示当前场景图内所有的热点，可以点击热点标题弹出热点配置界面进行编辑，也可以通过点击场景图上的热点进入热点配置界面

  5. # 沙盘配置

    沙盘是指整个工程的平面俯瞰图，当工程存在多个位置点，需配置多张场景图时，使用沙盘功能可以快速找到自身定位以及让使用者对整个工程场景有一个清晰明了的认知

    # 背景

    背景是指沙盘的地图，这里可以设置图片的内容和标记点

    • 选择图片：点击 选择图片 后在素材库图片类型中选择一张图片，该图片作为沙盘图将会出现在工程场景的右上方；点击 重置 按钮会将选择的图片移除

    • 增加标记点：点击 增加标记点 ，界面弹出该工程所有的场景图，选择其中一个场景图后，在沙盘图的中间会出现一个标记点，可以移动该标记点配置位置以及点击该标记点配置视角（点击标记点后会出现一个圆圈，拉动圆周长上的红点可以配置目标场景初始视角在沙盘上的方向）

    # 标记点列表

    标记点列表上面显示沙盘上所有配置的标记点，所有的标记点都是以场景图名称命名，点击列表上的任意标记点相当于在沙盘上点击该标记点进行视角方向配置，也可以点击标记点名称后的删除图标移除该标记点

# 四、保存工程

工程配置完毕之后需要点击 保存 按钮确保配置的内容成功保存在数据库中，否则数据将会丢失；工程第一次保存时，界面会弹出工程命名的弹窗，填写工程名称后会自动修改基础配置中的工程名称。（仅在工程第一次保存时出现）

提醒：账号的登录具有时间限制，过长时间的登录将会自动退出账号，请随时进行保存尤其在完成内容配置之后

# 五、预览工程

在工程完成配置后，需要先进行 保存 操作后点击 预览 ，界面上会弹出一个预览窗口可以进行正常的观览，用于配置人员根据实际工程显示效果进行调试审查。

注意：预览弹窗中不会显示沙盘功能
