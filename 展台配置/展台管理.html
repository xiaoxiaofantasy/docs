# 展台管理

# 一、新增展台

点击页面右上方 添加展台 ，界面会出现展台信息配置弹窗，填写相关内容完成展台配置

# 必填选项

  • 展台名称：编辑展台名称用于查询和识别
    注意：多个展台的名称可以重复

  • 选择场景：选择场景即选择工程，下拉列表中会出现所有的已创建保存好的工程名称，选择其中的一个进行配置
    注意：一个展台只能选取一张场景（工程），但多个展台可以选取相同的场景（工程）

# 选填选项

  • 封面图片：可以上传图片作为展台管理界面上新展台的图标显示，增加辨识度，如果没有上传则使用系统默认的图片作为图标显示

  • 语言选择栏：该功能模块可以对工程展示配置语言选择

  1. 该功能栏中包含多种语言，可将语言栏中语种编辑为开启状态，则在观览该展台的工程场景上会出现语种切换按钮，该按钮可以选择已在展台配置中开启的语种作为观览工程的基础语言

  2. 语种改变工程中点位名称，场景语音播报内容，热点语音播报内容以及场景内基础设置的显示文字

  3. 所有的展台默认的语种为中文，当对该项不进行配置时（所有的语种处于关闭状态），工程场景中以中文作为语言显示信息

  • 智能功能栏：该功能模块可以对工程展示配置智能功能选择，包括下列四项

  1. 留言：开启该功能，场景上可通过手机号登录对后台进行留言操作

  2. 咨询：开启该功能，场景上可通过手机号登录对后台进行智能问答操作

  3. 直播：开启该功能，场景上可通过手机号登录观看后台配置的直播

  4. 洽谈室：开启该功能，场景上可通过手机号登录对后台进行会议室预约申请

# 二、编辑展台

新建成一个展台之后，点击展台显示图标上的 编辑 按钮，页面跳转到场景编辑中，并且会自动打开该展台配置的场景工程，可以进行工程上的编辑

# 三、配置展台

点击展台图标上 配置 按钮，页面会弹出该展台配置信息的窗口（该窗口与新建展台时弹出的展台信息配置窗口一致），在配置弹窗中所有的信息均可修改，点击确定后保存修改内容

# 四、预览展台

点击展台图标上 预览 按钮，页面弹出该展台工程的网址二维码图片和展示工程网址链接，可以通过手机扫描二维码或pc端访问浏览地址的方法进行观览

# 五、展台上下架

展台图标上的上架（下架）按钮，用于配置方控制展台工程开关状态

注意：所有的展台在新建完成后，默认为下架状态，需要手动点击 上架 按钮开启站台工程显示

# 六、查询展台

在展台名称查询框中输入所想查询的展台名称的关键词，点击查询按钮或者Enter键进行查询

# 七、删除展台

点击展台显示图标右上角的删除按钮可进行删除操作，或者点击图标左上角选中按钮后点击页面出现的 删除所选 按钮进行删除操作

点击页面 全选 按钮后可以进行删除所选操作，用于批量删除展台数据
