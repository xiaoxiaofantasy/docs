# 布局模板

# 一、新增布局

在布局模板界面点击 添加模板 按钮，界面会弹出新建模板的信息输入框

# 必填信息

  • 模板名称

    将你创建的模板集合命名，用于查询和使用

  • 素材上传

    上传布局模板图模块分为 全景图上传 和 六面体上传

  1. 全景图上传

    全景图大小格式要求：

推荐上传的全景图分辨率在 5000*2500 以上（更低的低分辨率影响体验感官）

推荐全景图长宽比为 2：1（若全景图的比例不为标准，上传之后可能会有黑色区域填充影响体验感官）

上传全景图的格式可为 PNG, JPG...

  1. 六面体^1上传

    六面体大小格式要求：

六面体必须一次性上传6张图片或以6的倍数上传，每6张图片视为一个整体

六面体的图片长宽比为 1：1

六面体图片的命名规则 image_l( f ,r ,b ,u ,d ).png/jpg 。其中image可以是您任意对图片的命名，此外 l ，f ，r ，b ，u ，d 表示该图片所在的方位分别为 左 ，前 ，右 ，后， 上，下【您可以将自己想象在一个正方体内部，方位就是正方体的每个面与您的视角所形成的位置关系】

上传六面体图片的格式可为 PNG, JPG...

# 选填信息

  • 模板尺寸

    控制上传模板的尺寸大小，当该选项没有赋值时，使用上传图片的原本尺寸大小，填写的尺寸太小会出现在模板图标的左下角

  • 模板封面图

    可以上传图片作为布局模板界面上新布局的页面显示，增加辨识度，如果没有上传则使用系统默认的图片作为页面显示

# 二、查询

在模板名称查询框中输入所想查询的模板名称的关键词，点击查询按钮或者Enter键进行查询

# 三、编辑

对已新建的布局模板进行编辑，可以编辑模板名称，模板封面图，模板尺寸，全景图删除和上传

# 四、移除

对已新建的布局模板进行移除（删除）操作，不可恢复

# 五、全选

点击布局模板界面左上角的全选按钮，所有当前页面的布局模板处于选中状态，右侧出现移除按钮可以一键移除当前页面所有的模板
